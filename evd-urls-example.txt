# ===============================================
# EasyVideoDL – URL List for Batch Download Mode
# ===============================================
#
# HOW TO USE THIS FILE:
#
# 1. Enter ONE video or playlist URL per line.
#    Example:
#       https://youtube.com/first-video
#       https://youtube.com/second-video
#
# 2. Blank lines are allowed and ignored.
#
# 3. Lines beginning with "#" are comments and ignored.
#
# 4. URLs must be plain text — no quotes, no commas.
#
# 5. Playlists:
#       You can mix normal videos and playlists.
#       Example:
#         https://youtube.com/watch?v=abc123
#         https://youtube.com/playlist?list=XYZ789
#
# 6. The program will process URLs in the order they appear.
#
# 7. If you want to disable a URL temporarily, just put "#" in front of it.
#
